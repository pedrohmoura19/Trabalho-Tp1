Detailed Description

Código destinado a armazenar o CPF.

O CPF (Cadastro de Pessoa Fisica) válido informado deve ser no formato XXX.XXX.XXX-XX em que X é um dígito de 0 a 9.Cada CPF inclui dígitos verificadores calculados por meio de algorítimo apropriado.
