Detailed Description

Código destinado a armazenar o CEP.

Código válido caso o valor esteja entre as seguintes faixas:

-1000000 a 5999999 e 8000000 a 8499999 : São Paulo

-20000000 a 26600999 : Rio de Janeiro

-70000000 a 70999999 : Brasília. -40000000 a 41999999 : Salvador. -60000000 a 60999999 : Fortaleza.
