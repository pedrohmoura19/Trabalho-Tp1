Detailed Description

Código destinado a armazenar o Emissor.

O Emissor é um texto composto por 5 a 30 caracteres onde cada caracter pode ser letra (A-Z ou a-z) ou espaco, há pelo menos 5 letras, não há espacos em sequencia e a primeira letra de cada termo é letra maiúscula. Esse texto representa o emissor do produto do investimento desejado.
